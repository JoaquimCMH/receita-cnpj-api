# RECEITA-CNPJ-API

> Este repositório fornece meios, através do docker, para facilitar o uso dos dados  do [Cadastro Nacional da Pessoa Jurídica (CNPJ)](http://www.receita.economia.gov.br/orientacao/tributaria/cadastros/cadastro-nacional-de-pessoas-juridicas-cnpj/dados-publicos-cnpj) oriundos da Receita Federal (RF).
